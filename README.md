**File README untuk Project Library Management System Python**
===========================================================

**Tentang Proyek**
-----------------

Library Management System adalah sebuah aplikasi yang dirancang untuk mengelola koleksi buku di sebuah perpustakaan. Aplikasi ini dibuat menggunakan bahasa pemrograman Python dan dapat diintegrasikan dengan database untuk menyimpan dan mengelola data buku.

**Fitur**
---------

*   **Pengelolaan Buku**:
    *   Menambahkan buku baru
    *   Menghapus buku
    *   Mengupdate buku
*   **Pencarian Buku**:
    *   Mencari buku berdasarkan judul
    *   Mencari buku berdasarkan pengarang
*   **Laporan**:
    *   Melihat daftar buku yang tersedia
    *   Melihat daftar buku yang dipinjam

**Instalasi**
------------

Untuk menginstal Library Management System, silakan ikuti langkah-langkah berikut:

1.  **Membuat Environment Python Baru**
    *   Buka terminal dan jalankan perintah `python -m venv library-management-system`
    *   Aktifkan environment baru dengan perintah `source library-management-system/bin/activate`
2.  **Menginstal Dependensi**
    *   Jalankan perintah `pip install -r requirements.txt`
3.  **Mengatur Database**
    *   Konfigurasi database di file `config.py`
4.  **Menginstal Aplikasi**
    *   Jalankan perintah `python main.py`

**Contoh Penggunaan**
--------------------

Berikut adalah contoh penggunaan Library Management System:

1.  **Menambahkan Buku Baru**
    *   Jalankan perintah `python main.py add_book`
    *   Masukkan judul, pengarang, dan tahun terbit buku
2.  **Mencari Buku**
    *   Jalankan perintah `python main.py search_book`
    *   Masukkan judul atau pengarang buku yang ingin dicari
